# **App Name**: QR Bistro

## Core Features:

- QR Code Menu Access: Customers scan a QR code to access the menu for their specific table.
- Menu Display: Display categories and food items with price and image.
- Cart Management: Customers can add/remove items from their cart and view the total bill.
- Order Placement: Customers place orders, receiving a confirmation message after successful placement.
- Admin Dashboard: Secure login for admin to manage menu items and categories.
- Order Management: Admin can view incoming orders and update their status.
- Kitchen Dashboard: Real-time list of new orders for the kitchen staff with order status updates. The tool decides when to generate an audible notification sound for new orders.

## Style Guidelines:

- Primary color: A warm, inviting orange (#FF7043) to evoke a sense of appetite and energy.
- Background color: Light cream (#FFF3E0) for a clean, appetizing backdrop.
- Accent color: Muted gold (#FFB300) to highlight key interactive elements such as 'Add to Cart' buttons and order status updates.
- Body and headline font: 'PT Sans', a humanist sans-serif that is readable in both headings and body text, giving a friendly, modern feel.
- Use minimalist icons to represent food categories and order status, ensuring clarity and ease of use.
- Mobile-first responsive design using modern cards for food items to maintain a simple, restaurant-style interface across all devices.
- Subtle animations and transitions for order status updates and menu item interactions to enhance the user experience.
